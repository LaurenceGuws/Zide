# Review Archive

This folder holds completed review snapshots, rollout records, and evidence that
still matter as historical context but should not read like current authority.

Rules:

- Keep only documents whose main value is rationale, migration history, or
  before/after evidence.
- Do not surface these files in top-level docs navigation unless a current doc
  explicitly needs one as supporting evidence.
- Current authority belongs in `docs/` and active `app_architecture/` design or
  todo docs, not here.
