# Active Work Queues

This folder holds active implementation queues, migration checklists, and
modularization plans.

Use it for:

- outstanding work
- implementation sequencing
- migration checkpoints
- subsystem-specific task state

Do not use it for:

- stable architecture authority
- historical investigations
- public-facing project overview

Ownership split:

- `docs/todo/` — active work queues and implementation plans
- `app_architecture/` — current architecture, boundaries, and technical design
- `app_architecture/review/` — historical audits, investigations, and archived
  rollout evidence
